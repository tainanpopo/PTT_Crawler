# PTT_Crawler
檔案說明 :

* runtime.txt : 要使用的語言
* requirements.txt : 要安裝的package
* Procfile : 告訴Heroku你要的服務是甚麼
* ptt.py : 主程式

基本操作 :

* user打指令 : Gossiping
* bot推送 : 八卦版最新一篇文章
* user打指令 : Beauty
* bot推送 : 表特版最新一頁的隨機一張圖片
* user打指令 : NBA
* bot推送 : NBA版最新一篇文章
